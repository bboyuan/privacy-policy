PaperMode Privacy Policy

Last updated: April 27, 2026

PaperMode is a Chrome extension that applies a paper-inspired visual reading mode to webpages.

Data Collection

PaperMode does not collect, sell, transmit, or share personal data.

PaperMode does not collect:

  • Browsing history
  • Page content
  • Search history
  • Account information
  • Names, email addresses, or contact information
  • Location data
  • Financial, health, or authentication information
  • Analytics or telemetry

Local Extension Settings

PaperMode stores extension settings so the extension can remember whether it is enabled, which mode is selected, the selected intensity, and optional per-site overrides by hostname.

These settings are stored with Chrome's storage.sync API. Chrome may sync these settings across the user's signed-in Chrome browsers through the user's Google account. PaperMode's developer does not receive or access these settings.

Permissions

PaperMode requests only the permissions needed to apply and configure the visual reading mode:

  • storage: saves the user's PaperMode settings
  • activeTab: allows PaperMode to work with the current tab after user interaction
  • scripting: injects PaperMode scripts when needed
  • http://*/* and https://*/* content script matches: applies the visual reading mode on regular webpages

Third Parties

PaperMode does not use analytics providers, advertising networks, tracking pixels, remote logging, or third-party data processors.

Contact

For privacy questions, contact PaperMode through GitHub Issues.
